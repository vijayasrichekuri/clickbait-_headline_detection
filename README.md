🧠 Clickbait Headline Detection System
This project is an AI-based web application that detects whether a news headline is clickbait or genuine using Machine Learning and Natural Language Processing (NLP).
The system analyzes headlines using TF-IDF text features and predicts results through a trained model built with scikit-learn.
The interface is developed using Streamlit.
🚀 Features :
Detects Clickbait vs Non-Clickbait headlines
Interactive Streamlit AI dashboard
Prediction probability graph
Word Cloud visualization
Confusion Matrix for model evaluation
🛠 Technologies Used :
Python
Streamlit
scikit-learn
Pandas
Matplotlib
Seaborn
▶ Run the Project
Install dependencies:
pip install streamlit scikit-learn pandas matplotlib seaborn wordcloud joblib
Run the app:
streamlit run app.py
Open in browser:
http://localhost:8501
